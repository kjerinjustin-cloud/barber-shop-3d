# J S Ramesh Hair Salon - 3D Interactive Website

A complete 3D interactive barber shop website with just 5 files!

## Files

1. **index.html** - Main website with all sections
2. **style.css** - Complete styling
3. **script.js** - 3D scene & interactions
4. **package.json** - Project metadata
5. **README.md** - Documentation

## Features

✨ **Interactive 3D Salon** - Drag to rotate, scroll to zoom  
🛎️ **8 Services** - With pricing  
📅 **Online Booking** - Date & time selection  
📸 **Gallery** - Photo showcase  
📞 **Contact Info** - Full details  
📱 **Responsive Design** - Works on all devices  

## Live Demo

👉 https://barber-shop-3d.vercel.app

## How to Use

### Online (Easiest)
1. Just open the live link above
2. Done! 🚀

### Local Development
1. Clone or download the files
2. Run: `python -m http.server 3000`
3. Open: http://localhost:3000

## Customization

- **Contact Info** - Edit in index.html
- **Colors** - Change in style.css (look for #d4a574)
- **Services** - Add/remove in index.html services section
- **Gallery Images** - Replace image URLs
- **3D Models** - Enhance in script.js

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- Three.js (CDN)
